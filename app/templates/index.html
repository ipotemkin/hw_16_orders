Orders API

Functions:

USERS:

  • GET /users – get all users
  • GET /users/ID – get a user with ID
  • POST /users/ID – add a new user
  • PUT /users/ID – update a user with ID
  • DELETE /users/ID – delete a user with ID

ORDERS:

  • GET /orders – get all orders
  • GET /orders/ID – get an order with ID
  • POST /orders/ID – add a new order
  • PUT /orders/ID – update an order with ID
  • DELETE /orders/ID – delete an order with ID

OFFERS:

  • GET /offers – get all offers
  • GET /offers/ID – get an offer with ID
  • POST /offers/ID – add a new offer
  • PUT /offers/ID – update an offer with ID
  • DELETE /offers/ID – delete an offer with ID
